About the app

iCook is an app that can be used for students with any range of culinery skills. As this is a community app, there will always be something new to tantislise your tastebuds! Register for a free account today.

How to use the app

In order to use the 'iCook' application you will need to register an account. It's all free just type your details using the register form make sure you haven't created an account for our app before as you will be unable to register with email address which has already been used.
